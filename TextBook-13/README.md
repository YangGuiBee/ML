# [2] 평균 제곱 오차 (Mean Squared Error, MSE)
선형 회귀 모델에서 예측값과 실제값의 차이를 제곱한 것들의 평균
$MSE = (1/(n-k-1)) * ∑(i=1 to n) (yi - ŷi)^2$ 
(n : 샘플 데이터의 개수, k : 독립 변수의 개수, yi : 실제값, ŷi : 모델의 예측값)
MSE가 작을수록 모델의 예측 성능이 높다고 해석할 수 있습니다.
