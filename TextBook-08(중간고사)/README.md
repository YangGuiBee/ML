<br>

기본 원리를 설명하고, 두 방법의 차이점을 비교하시오.<br>
아울러, 연구 목적이나 데이터 특성에 따라 두 방법 중 하나를 선택해야 하는 상황을 하나 제시하고, 그 이유를 설명하시오.<br>

<br>

비지도학습 결과를 평가하는 것이 지도학습보다 어려운 이유를 논하시오.<br>
